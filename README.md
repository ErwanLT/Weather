# EWeather - 1.5.0-SNAPSHOT
A simple web application using spring-boot to call the DarkSky Api and dispatch the result in a beautiful (all is relative) web page 

### Environment setup
Currently using
  - IntelliJ 2018.2
  - java 1.8.0_181
  - Maven 3.5.4
  - Lombok plugin
