Formulaire de recherche

Vous pouvez chercher la météo pour n'importe quel endroit autour du globe (oui, la terre est ronde, n'en déplaise à certains)
Pour ce faire vous pouvez remplir le champs de recherche avec :

  • la rue
  • la ville
  • l'état
  • le pays
  • le code postal
Attention, plusieurs ville peuvent avoir le même code postal
Exemple :
Pour le code postal 75000, je peux avoir Paris en France et Residencial Antônio Carlos Pires au Brésil
Dans des cas comme celui-ci, il vaut mieux renseigner l'adresse complete.
